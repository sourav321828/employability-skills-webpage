Employability Skills Webpage

This project is an Employability Skills Webpage developed as part of the module assessment (Task 3). The purpose of this webpage is to demonstrate the development of key employability skills through practical module activities.

The webpage highlights the following skills:

Problem Solving
Analytical Thinking
Communication
Teamwork
Time Management

Each skill is supported with evidence collected during the project, including screenshots of analysis work, presentation slides, Work Breakdown Structure (WBS), Gantt chart, and RACI matrix.

The webpage also includes a personal reflection that explains how these skills were developed and improved throughout the module. This reflection is written in the first person and meets the requirement of demonstrating learning outcomes.

The website is built using HTML, CSS, and JavaScript, with interactive features such as smooth scrolling, hover effects, and responsive design to enhance user experience.
